# GraphNet scratch

## Описание: Расширение для работы с моделями, скомпилированными с использованием GraphNet

## Запуск(для разработчиков/тестировщиков):
- Выполнить команду `npm install` в обеих директориях
- Выполнить команду `npm run watch` в директории 'scratch-vm'
- Выполнить команду `npm start` в директории 'scratch-gui'
- Scratch будет активен по адресу `localhost:8601`
